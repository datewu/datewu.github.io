总的说来敲了100来条指令，比较重要的是下面三条指令：

  1. sudo -i切换为root用户；
  2. yumdownloader 保存通过yum安装的rpm包 (配合yum -C install 和SCP可以加快LAN下所有主机的安装速度)；
  3. rpm -qc 查询安装包配置信息;

timeout

默认情况下，Azure 的Linux主机的sshd 服务会把闲着超过1分钟的客户端踢下线ssh session timeout。 修改本地ssh 客户端连接配置添加ServerAliveInterval等配置项就可以规避上面说的timeout限制：

#### .ssh/config snippet
HOST singapore2
  HostName 40.50.60.70
  IdentityFile ~/.ssh/az
  ServerAliveInterval 120
  ServerAliveCountMax 30
  ConnectTimeout 30
