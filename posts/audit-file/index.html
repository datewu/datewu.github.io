审计目录

使用ausearch查看auditd系统日志，审计文件和目录

Jun 20, 2018
阅读时长: 3 分钟

今天调试容器应用的时候发现，app运行一段时间后，容器外挂的一个volumn会偶发性的被删除。 于是需要监控下到底是谁/哪个进程把文件目录给删除了。

google一阵子后，发现可以使用auditd 服务来监控和搜索出都有那些进程操作够目标文件/目录。

整个过程分为3步：

  1. 开启 auditd 服务；
  2. 使用auditctl 配置 auditd服务；
  3. 一段时间之后 使用 ausearch 来查看/搜索审计的日志。

启动监控

开启auditd服务：

systemctl start auditd
## you may need `mkdir /var/log/audit`

添加监控规则

编辑审计规则：

## list existing rules
auditctl -l

## clean existing rules
auditctl -D

## watch /var/run/yourfolder
auditctl -w /var/run/yourfolder -p war -k serachkey
